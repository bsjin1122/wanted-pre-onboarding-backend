# wanted-pre-onboarding-backend
프리온보딩 백엔드 인턴십 선발과제

> Java 21 & Springboot(3.3.2), MySQL

### 커밋 메시지 포맷
- `feat`: 새로운 기능 추가
- `fix`: 버그 수정
- `docs`: 문서 수정
- `style`: 코드 포맷팅, 세미 콜론 누락 등 코드 자체의 변경이 없는 경우
- `refactor`: 코드 리팩토링
- `test`: 테스트 추가나 수정
- `chore`: 빌드 태스크 업데이트, 패키지 매니저 설정 등 기타 변경
